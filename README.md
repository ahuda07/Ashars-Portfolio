# Ashar Huda Portfolio

Personal portfolio for Ashar Huda, a UCI Software Engineering and Informatics
student focused on software engineering, AI engineering, backend systems, and
full-stack applications.

Built with Next.js, TypeScript, Tailwind CSS, Radix UI, Lucide Icons, and Next
Themes.

## Running Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run lint
npm run build
```
